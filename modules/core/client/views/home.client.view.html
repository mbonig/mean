STARZ website boilerplate!

Congrats! You've created a new STARZ website.

Now that you have this running you need to make some changes for this specific project.

  • Edit the /config/env/default.js file. Replace:
    • app.title
    • app.description
    • app.keywords
    • app.googleAnalyticsTrackingID
    • sessionSecret
    • mailer.from
    • mailer.sendgrid.templates - go to SendGrid and create new forgot/reset email templates.
  • Edit the /config/env/development|production.js file, replace:
    • db.uri - replace db name with short-name-dev (or short-name). E.g. media-room-dev|media-room
  • Edit the /package.json, replace:
    • name
    • description
    • repository.url
  • Edit the /bower.json, replace:
    • name - should match package.json
    • description - should match package.json
  • Edit the /modules/core/client/app/config.js, replace "mean" with your application short-name. E.g. media-room
