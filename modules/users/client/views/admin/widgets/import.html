Import Users

This allows you to bulk create new Users in the system. You will provide a CSV file with the required information.

The format is very specific. Please download the template when this is a completely new list.

Click here to download the current Group's sample. Using this and then filling in the information as you go will greatly reduce the change of errors!

Please make sure you provide the following information for each new user. The header row must have the values listed below but they can be in any order.

  • Organization
  • First Name
  • Last Name
  • Email Address
  • Job Title - optional
  • Group - leave the exact value that's in the template.
  • Notes
  • Username
  • Password - optional, if you leave it blank one will be generated. If this is an existing user the password will be ignored.

Any fields not marked as optional are required.

No emails will go out to new users. You are responsible for making any new users aware of their username and password.
